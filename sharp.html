Visual J Sharp

Visual J# (poznatiji kao J#) je set programskih alata koji dozvoljavaju korisnicima da koriste programski jezik Java za pisanje aplikacija, odnosno programa, koji će se pokretati na Microsoft .NET platformi* .
Kako Microsoft ne podržava "Java Virtual Machine"* kao deo .NET-a, J# obezbeđuje programerima kojima je bliži koncept i sintaksa Jave da pišu i kompajluju program koji će se lako pokretati na .NET Framework-u.
Izvorni kod je konvertovan u Microsoft Intermediate Language (MSIL)*, koji je ekvivalentan Javinom bajtkodu. Program koji je već kompajlovan u Javinom bajtkodu može biti konvertovan u MSIL.




  • Korišćenjem Visual J#, poznavaoci Jave mogu uraditi sledeće:
  • iskoriste znanje Java programiranja za razvoj .NET aplikacija
  • podržava JDK 1.1.4 biblioteku i većinu JDK 1.2 biblioteka klase java.util
  • koristi proširenja na programskom jeziku Java za svojstva i događaje
  • pretvori Visual J++ projekte u J# pomoću Project Import Wizard-a
  • kreira dokumentaciju zasnovanu na Javadoc
  • omogući podršku apleta, poput modela primene J# Browser Controls

    • Sa Visual J#, Microsoft uključuje njihove standardne biblioteke
    koje su ekvivalentne sa bibliotekama Java Development Kit-a*
    verzije 1.1.4. J# podržava funkcionalnost Visual J++ i Microsoft ekstenzija.
    Kod pisan u J# imaće ekstenziju .jsl.
